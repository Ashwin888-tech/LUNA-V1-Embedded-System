# LUNA V1 – Embedded System

## Overview

LUNA V1 is the first-generation semi-humanoid robot developed by the TechBots team. The project integrates robotics, embedded systems, artificial intelligence, computer vision, and voice interaction into a single interactive robotic platform.

This repository contains **only the embedded systems firmware** developed for the robot. The firmware is responsible for controlling the robot's locomotion, servo movements, gesture execution, and communication with the hardware peripherals.

The AI software, computer vision modules, and voice interaction components were developed separately by other members of the project team.
